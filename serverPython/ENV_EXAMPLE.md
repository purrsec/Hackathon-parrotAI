# Variables d'environnement

Créez un fichier `.env` à la racine du dossier `python_controller` avec :

```env
# IP du drone (Sphinx/ANAFI)
DRONE_IP=10.202.0.1

# Configuration du serveur
API_HOST=0.0.0.0
API_PORT=8000

# Niveau de logging (DEBUG, INFO, WARNING, ERROR)
LOG_LEVEL=INFO
```

Ou modifiez directement `config.py` pour les valeurs par défaut.
